MonCvPro est un site web permettant de créer et afficher un CV en ligne une fois que l'utlisateur remplit les informations

l'architecture du projet est :un dossier assets qui contient 3 autres dossiers tels que "script.js", "style.css", "images" et sur la racine y'a l'index.html.

nous avons utlise html, css et javascript .

Le travail a été réparti comme suit : abdoulaye : création de la structure HTML et du formulaire CV Massamba: lui il a gere le figma alassane : il a gerer les templates.

j'ai eu quelque difficulte pendant la creation de ce site surtout quand l'utlisateur doit appuver sur "creer" pour generer automatiquement un cv qui etait un peu difficile mais on a gerer.
